# Caption System

> Documents the FactsFactory caption pipeline.
> Code: `scripts/production/ass_captions.py`
> Last updated: 2026-04-09

---

## Overview

Captions are generated by `ass_captions.py` and burned directly into the final video.
The process runs automatically at the end of `main.py` for every style.

```
voiceover MP3
     │
     ▼
Whisper (word timestamps)
     │
     ▼
Claude Haiku — 8-category semantic analysis
     │
     ▼
ASS file generation (4-tier emphasis system)
     │
     ▼
ffmpeg ass= filter — burned into final video
```

---

## Public API

```python
from scripts.production.ass_captions import generate_ass_captions, burn_ass_captions

# 1. Transcribe + analyze + generate .ass file
ass_path = generate_ass_captions(
    audio_path="inbox/ID_voice.mp3",
    output_dir="logs/captions/",
    video_id="my-video-001",
    script_text="Full narration text...",   # optional but improves quality
)

# 2. Burn into video (writes to a temp file then replaces to avoid input=output conflict)
burn_ass_captions(
    video_path="output/my-video-001_raw.mp4",
    ass_path=ass_path,
    output_path="output/my-video-001_final.mp4",
)
```

## 4-Tier Emphasis System

| Tier | Scale | Color | Use |
|---|---|---|---|
| 0 — neutral | 100% | White | Most words |
| 1 — mild | 110% | White | Function words in hook, end words |
| 2 — strong | 122% | Yellow | Danger, action, payoff, key words |
| 3 — climax | 140% | Yellow | Punchline word (one per script) |

---

## Claude Haiku Analysis (8 categories)

The script is analyzed for:
- `key_words` — high emotional weight
- `danger_words` — threat/risk language
- `action_words` — rescue/response verbs
- `payoff_words` — resolution/safety language
- `turn_words` — pivot words (but, then, suddenly)
- `cta_words` — engagement words (follow, share)
- `punchline_word` — single most climactic word
- `hook_end_word` — last word of the first sentence

Without ANTHROPIC_API_KEY, local keyword sets are used as fallback.

## CLI Usage

```bash
# Generate ASS + burn into video in one step
python scripts/production/ass_captions.py \
  --audio inbox/ID_voice.mp3 \
  --video output/ID_raw.mp4 \
  --video-id ID \
  --output-dir output/ \
  --script "Full narration text here..."
```

Output: `output/{video-id}_final.mp4` (captions burned in)
ASS file cached at: `logs/captions/{video-id}.ass`
